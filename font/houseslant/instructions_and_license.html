A WEB FONT FROM HOUSE INDUSTRIES

Here is our recommendation for code to use in your css stylesheet when using the House Slant font. Just copy and paste this code into your style sheet, and apply to desired rule, as shown below applied to an h3 tag.

@font-face {
    font-family: "House Slant";
    src: url('HouseSlant.eot');
    src: url("HouseSlant.eot?#iefix") format("embedded-opentype"),
         url("HouseSlant.woff") format("woff");
    font-style: normal;
    font-weight: 400;
    font-stretch: normal;
}

When you use the @font-face as noted above, the code below will yield the following results:

HOUSE SLANT IS MY FAVORITE WEB FONT!

h3 {
    font-family: "House Slant", script;
    text-rendering: optimizeLegibility;
    font-weight: normal;
}

*******Notes*******

This assumes that the fonts reside in the same directory as the CSS file on your server. If you place the fonts in a different directory, that location must be included in the "src:" attribute.

We recommend using the "text-rendering:optimizeLegibility;" attribute. To avoid issues with Google Chrome, please do not use negative word spacing of more than 0.8 em in combination with optimizeLegibility.

For more information on using @font-face, please consult w3 schools web site.
